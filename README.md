## 📘 2. README для HealthTracker (iOS-трекер питания и тренировок)


# 💪 MyGainApp — трекер питания и тренировок для iOS

**HealthTracker** — это мобильное приложение для отслеживания калорий, тренировок и прогресса в достижении целей.  
Всё хранится локально — без интернета, с уведомлениями и статистикой.

---

## 🎯 Функционал

- ✅ Дневник питания (добавление блюд, подсчёт КБЖУ)
- ✅ Трекер тренировок (вид, длительность)
- ✅ Графики прогресса (неделя / месяц)
- ✅ Сохранение данных в Core Data
